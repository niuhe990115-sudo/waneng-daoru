# 测试指南

## 测试步骤

请在浏览器中打开 http://localhost:3000，然后按以下步骤测试每个文件：

### 测试文件列表

1. **template1-standard.xlsx** - 标准格式
2. **template2-ecommerce.xlsx** - 电商格式
3. **template3-english.xlsx** - 英文格式
4. **template4-grouped.xlsx** - 分组格式
5. **template5-multisheet.xlsx** - 多工作表格式

### 每个文件的测试流程

对于每个文件，请执行以下操作：

1. **点击"重新开始"按钮**（如果是第一次测试则跳过）

2. **上传文件**
   - 从 `test-data` 文件夹选择一个文件
   - 拖拽或点击上传

3. **点击"AI 生成规则"按钮**
   - 等待 AI 分析完成
   - 查看自动生成的规则

4. **点击"开始解析"按钮**
   - 查看解析结果

5. **检查控制台日志**（按 F12 打开开发者工具）
   
   应该看到类似以下的输出：
   ```
   [Excel解析] 开始解析，共 X 个工作表
   [Excel解析] 工作表 Sheet1 共有 XX 行
   [Excel解析] 前5行数据:
     第0行: ["外部编码", "发件人姓名", ...]
     第1行: ["...", "...", ...]
   [findCol] ✓ 完全匹配: "外部编码" -> 第0行第0列
   [findCol] ✓ 完全匹配: "收件人姓名" -> 第0行第4列
   [findCol] ✓ 完全匹配: "收件人电话" -> 第0行第5列
   [解析第X行] externalCode="XXX", recipientName="XXX", ...
   [Excel解析] 总共解析出 X 条订单
   ```

6. **验证解析结果**
   - 检查所有字段是否正确填充
   - 特别关注：
     - 外部编码
     - 收件人姓名、电话、地址
     - SKU物品名称和数量

7. **修正错误**（如果有）
   - 如果字段为空或错误，查看控制台日志找出原因
   - 可能需要调整 AI 生成的规则

8. **提交订单**
   - 确认所有数据正确后，点击"提交订单"
   - 应该看到成功提示

9. **检查订单是否保存**
   - 切换到"已导入运单"标签页
   - 应该能看到刚才提交的订单记录

## 常见问题排查

### 问题1：所有列都返回 -1（未找到）

**症状**：
```
[findCol] ✗ 未找到列: "外部编码"
```

**可能原因**：
- Excel 中的列名与规则中的 sourceField 不匹配
- 列名包含引号或其他特殊字符

**解决方案**：
- 查看 `[Excel解析] 表头行` 的输出，确认实际列名
- 手动编辑规则，修改 sourceField 使其与实际列名一致

### 问题2：所有字段都指向同一列

**症状**：
```
[findCol] ✓ 找到列: "外部编码" -> 第2行第10列
[findCol] ✓ 找到列: "收件人姓名" -> 第2行第10列
```

**可能原因**：
- findCol 函数的包含匹配太宽松

**解决方案**：
- 已修复为两阶段匹配策略（优先完全匹配）

### 问题3：JSON 解析失败

**症状**：
```
Unexpected token '<', "<!DOCTYPE "... is not valid JSON
```

**解决方案**：
- 系统已添加自动重试机制（最多重试2次）
- 如果仍然失败，尝试重新点击"AI 生成规则"

### 问题4：订单提交成功但没有保存

**解决方案**：
- 已修复演示模式的内存存储
- 订单现在会保存在内存中，可以在"已导入运单"页面查看

## 调试技巧

1. **查看详细日志**
   - 打开浏览器控制台（F12）
   - 查看所有以 `[Excel解析]` 和 `[findCol]` 开头的日志

2. **对比不同文件**
   - 注意不同文件格式的差异
   - 观察 AI 如何针对不同格式生成不同的规则

3. **手动编辑规则**
   - 如果 AI 生成的规则不正确，可以点击"新建规则"手动调整
   - 或者在 AI 生成的基础上修改

4. **使用"查看规则"按钮**
   - 查看完整的 JSON 规则
   - 检查 fieldMappings 中的 sourceField 是否与 Excel 列名匹配
